# 📘 Assignment: Building REST APIs with FastAPI

## 🎯 Objective

Learn how to build fast and modern REST APIs using the FastAPI framework. You'll create an API with routing, request/response handling, and data validation to understand core web development concepts.

## 📝 Tasks

### 🛠️ Create a Basic FastAPI Application

#### Description
Build a FastAPI application that serves as a foundation for a REST API. Set up the project structure, install dependencies, and create HTTP endpoints that handle GET and POST requests.

#### Requirements
Completed program should:

- Install FastAPI and Uvicorn dependencies
- Create a main FastAPI application with proper initialization
- Implement at least 2 GET endpoints that return different data
- Implement at least 1 POST endpoint that accepts and processes data
- Include appropriate HTTP status codes for different responses
- Run the server successfully on localhost:8000

### 🛠️ Add Data Validation and Documentation

#### Description
Enhance your API with proper data validation using Pydantic models and explore FastAPI's automatic documentation features.

#### Requirements
Completed program should:

- Define Pydantic models for request/response data validation
- Use type hints throughout the application
- Implement input validation for POST requests
- Access the auto-generated API documentation at `/docs`
- Return proper error responses with validation messages
