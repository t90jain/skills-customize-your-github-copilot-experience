# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build the classic word-guessing game where players guess letters to reveal a hidden word before running out of attempts. You'll practice string manipulation, loops, conditionals, and random selection.

## 📝 Tasks

### 🛠️ Create the Hangman Game

#### Description
Implement a complete Hangman game that allows players to guess letters and reveals the hidden word progress. The game should randomly select words, track incorrect guesses, and determine win/lose conditions.

#### Requirements
Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses and display current progress in _ _ _ format
- Track the number of incorrect guesses remaining
- End when the word is guessed correctly or attempts are exhausted
- Display clear win/lose messages at the end of the game
